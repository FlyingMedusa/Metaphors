How does the Cognitive Metaphor Graph Generator work?

The 'mechanism' is quite simple:

  1. Please click 'Upload' on the navbar
  2. You will be moved to the upload page (who could expect that). Please upload an .xls file there. Requirements:
  3. What's important when creatng the Excel file:
    • Column A: phrase - free formatting
    • Column B: just a target domain
    • Column C: just a source domain
    • Column D: abstractness. Please format as follows:
      • A - an abstract domain; C - a concrete domain
      • First letter: target domain; Second letter: source domain
  4. After a successful upload, you will be redirected to the tables and graphs section
  5. The program will generate frequency data
