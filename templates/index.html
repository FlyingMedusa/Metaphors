How does the Cognitive Metaphor Graph Generator work?

The 'mechanism' is quite simple:

  1. Please click 'Login/Upload' on the navbar and enter your name
  2. You will be moved to the user page, upload a csv file there. Requirements:
    • Each row should contain a quote and a cognitive metaphor
    • A cognitive metaphor should be: X is Y, or a X is an Y, etc.
    • X and Y (metaphor domains) HAVE TO be in capital letters
  3. After a successful upload, you will be redirected to the graph-creation section
  4. The program will generate frequency data and a graph
